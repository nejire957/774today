このサイトについて

  • ななしいんくのVTuberの配信スケジュールをタイムライン形式で表示する非公式ファンサイトです。
  • 配信スケジュールは各タレントのYouTubeチャンネルとTwitchチャンネルから取得しています。他者のチャンネルでの配信や待機所が作られていない予定は表示されません。
  • 更新は1時間ごとに行っています。突発的な配信や長時間の配信は正しく表示されない場合があります。
  • スマートフォンアプリのように、ホーム画面から直接使うことができます。ブラウザのメニューから「インストール」を実行してください。(PWA対応)
  • 画面右上の右から2番目にあるボタンから表示するタレントを選択できます。選択内容はこのサイトを離れても維持されます。この機能のために、このサイトはブラウザのローカルストレージを使用します。
  • ご意見・ご要望は下記TwitterアカウントのDMで受け付けています。
  • 製作者:ねじれ図形 | X(Twitter) | GitHub

注意

  • このサイトはななしいんく所属タレントの二次創作です。開発にあたっては「ななしいんく」所属タレント二次創作ガイドラインに従っています。
  • このサイトはYouTube APIを使用しています。開発にあたってはYouTubeの利用規約と、YouTube APIの利用規約および開発方針に従っています。 このサイトを利用することは、上記の規約に同意することとみなされます。
  • このサイトはTwitch APIを使用しています。開発にあたってはTwitchのサービス利用規約とデベロッパー規約に従っています。
  • このサイトはGoogle Analyticsを使用しています。開発にあたっては、Googleアナリティクスポリシーに従っています。
  • ユーザーに関するデータの取り扱いについて
    • このサイトはユーザーからこのサイトへのアクセスに関するデータを収集します。データの収集にはcookieを使用します。
    • Googleは収集されたデータをプライバシーポリシーに従って管理・分析します。
    • サイト製作者はGoogleから分析結果を受け取り、サイトの利用状況を把握します。
    • サイト製作者は、収集したデータをGoogle Analytics上で14か月保持します。
    • サイト製作者は、下記の場合を除いて、個人を特定できる情報を他の企業、組織、個人と共有することはありません。
      • 法律上の義務に応じて、または法律上認められる範囲内で、一般の人々に害が及ぶことを防ぐため。
    • サイト製作者は、サイトの一般的な利用傾向などの、個人を特定できない情報を公開することがあります。
    • このサイトを利用することは、上記のユーザーに関するデータの取り扱いに同意を示すこととみなされます。
